1. 相関分析

相関係数とは？

2つの変数間の直線的な関係の強さと方向を表す指標です。-1から+1の値を取ります。

r = Σ[(xi - x̄)(yi - ȳ)] / √[Σ(xi - x̄)² × Σ(yi - ȳ)²]

Pearson相関係数

  • 線形関係の強さを測定
  • 正規分布を仮定
  • 外れ値の影響を受けやすい

Spearman相関係数

  • 順位相関（単調関係）を測定
  • 正規分布を仮定しない
  • 外れ値の影響を受けにくい

相関係数の解釈基準

相関係数の絶対値 関係の強さ 実務的な意味
0.9 〜 1.0 非常に強い相関 ほぼ完全な線形関係。多重共線性の可能性あり
0.7 〜 0.9 強い相関 明確な関連性あり。予測に有用
0.5 〜 0.7 中程度の相関 一定の関連性あり。他の要因も考慮必要
0.3 〜 0.5 弱い相関 わずかな関連性。単独では予測困難
0.0 〜 0.3 ほとんど相関なし 実質的に無関係

📝 実例

製品Aシェアと営業人員数の相関係数 = 0.72

→ 強い正の相関があり、営業人員数が多いほど製品Aのシェアが高い傾向がある

⚠️ 注意点

  • 相関は因果関係を意味しない
  • 第3の変数（交絡因子）の影響を考慮する必要がある
  • 非線形関係は検出できない

2. 回帰分析

回帰分析の基本

1つ以上の説明変数（X）から目的変数（Y）を予測する統計手法です。

Y = β₀ + β₁X₁ + β₂X₂ + ... + βₙXₙ + ε
β₀: 切片、β₁〜βₙ: 回帰係数、ε: 誤差項

R²（決定係数）

モデルが説明できる変動の割合

  • 0.8以上: 優秀
  • 0.5〜0.8: 良好
  • 0.5未満: 改善必要

調整済みR²

変数の数を考慮した決定係数

  • 変数を増やしても自動的に上がらない
  • モデル比較に適している

p値

係数が0である確率

  • p < 0.001: ***
  • p < 0.01: **
  • p < 0.05: *
  • p ≥ 0.05: 有意でない

RMSE

予測誤差の標準偏差

  • 元のデータと同じ単位
  • 小さいほど良い

回帰係数の解釈

例: 営業人員数の係数 = 0.35

→ 営業人員が1人増えると、製品Aシェアが0.35％増加する（他の変数が一定の場合）

単回帰と重回帰の使い分け：
  • 単回帰: 1つの要因の影響を単純に見たい場合
  • 重回帰: 複数要因の影響を同時に考慮したい場合

3. 多重共線性（Multicollinearity）

VIF（分散拡大係数）

説明変数間の相関が高すぎる状態を検出する指標です。

VIF = 1 / (1 - R²ᵢ)
R²ᵢ: 変数iを他の変数で回帰した時の決定係数

VIFの判定基準

VIF値 判定 対処法
VIF < 5 問題なし そのまま使用可能
5 ≤ VIF < 10 中程度の多重共線性 注意が必要、変数の削除を検討
VIF ≥ 10 深刻な多重共線性 変数の削除、主成分分析、Ridge回帰の使用

📝 実例

売上高のVIF = 15.3

→ 売上高は他の変数（従業員数など）と強く相関している

→ どちらか一方を削除するか、主成分分析を検討

4. 外れ値の検出と処理

IQR法（四分位範囲法）

データの中央50%の範囲を基準に外れ値を検出します。

外れ値の境界:
下限 = Q1 - 1.5 × IQR
上限 = Q3 + 1.5 × IQR
(IQR = Q3 - Q1)

Z-score法

平均からの標準偏差の倍数で外れ値を判定します。

Z = (x - μ) / σ
|Z| > 3 の場合、外れ値と判定

外れ値の処理方法

方法 説明 使用場面
キャッピング 外れ値を境界値に置換 データを保持したい場合
削除 外れ値を含む行を除外 明らかな異常値の場合
変換 対数変換などで正規化 分布が歪んでいる場合

⚠️ 注意点

  • 外れ値が重要な情報を含む可能性がある
  • ドメイン知識に基づいて判断する
  • 処理前後で結果がどう変わるか確認する

5. 交差検証（Cross-Validation）

K-fold交差検証

データをK個に分割し、K回の学習と検証を行う手法です。

5-fold交差検証の流れ

  1. データを5つに分割
  2. 4つで学習、1つで検証を5回繰り返し
  3. 5回の結果を平均化

結果の見方

例: R² = 0.75 (±0.08)

  • 平均R²: 0.75（モデルの性能）
  • 標準偏差: 0.08（安定性）
  • 標準偏差が小さい = 安定したモデル
交差検証の利点：
  • 過学習の検出
  • モデルの汎化性能の評価
  • 限られたデータの有効活用

6. 正則化回帰

Ridge回帰（L2正則化）

係数の2乗和にペナルティ

  • 全変数を使用（係数を0にしない）
  • 多重共線性に強い
  • 係数を縮小

Lasso回帰（L1正則化）

係数の絶対値和にペナルティ

  • 変数選択（係数を0にする）
  • 解釈しやすいモデル
  • スパースな解

Elastic Net

RidgeとLassoの組み合わせ

  • 両方の利点を活用
  • グループ化された変数に強い
  • バランスの取れた手法

α（正則化パラメータ）の選び方

  • 小さいα: 通常の回帰に近い（過学習のリスク）
  • 大きいα: 強い正則化（アンダーフィッティングのリスク）
  • 最適なα: 交差検証で決定

📝 Lassoで変数選択された例

元の変数: 30個 → 選択された変数: 8個

→ 8個の重要な変数だけで予測可能

→ シンプルで解釈しやすいモデル

7. 回帰診断

回帰の前提条件

  1. 線形性: XとYの関係が線形
  2. 独立性: 誤差項が独立
  3. 等分散性: 誤差の分散が一定
  4. 正規性: 誤差が正規分布

Shapiro-Wilk検定

残差の正規性を検定

  • p > 0.05: 正規分布
  • p ≤ 0.05: 非正規分布

Breusch-Pagan検定

等分散性を検定

  • p > 0.05: 等分散
  • p ≤ 0.05: 不等分散

Durbin-Watson統計量

自己相関を検定

  • < 1.5: 正の自己相関
  • 1.5〜2.5: 問題なし
  • > 2.5: 負の自己相関

Cook's Distance

影響力の大きい観測値

  • < 4/n: 問題なし
  • ≥ 4/n: 影響大

診断結果への対処法

問題 対処法
非正規性 変数変換（対数、Box-Cox）、ロバスト回帰
不等分散 加重最小二乗法、変数変換
自己相関 時系列モデル、差分を取る
影響力大の観測値 データの確認、外れ値処理

8. 分析結果の総合的な解釈

モデル選択の基準

AIC・BIC

モデルの良さを評価する情報量基準

  • 小さいほど良いモデル
  • AIC: 予測精度重視
  • BIC: シンプルさ重視

📊 総合的な判断例

分析結果サマリー
  • 調整済みR² = 0.72 → 良好な説明力
  • 交差検証R² = 0.68 (±0.05) → 安定した性能
  • VIF最大値 = 4.2 → 多重共線性なし
  • Durbin-Watson = 1.95 → 自己相関なし
  • 正規性p値 = 0.03 → やや非正規
結論

モデルは概ね良好。正規性の問題は軽微なため、このまま使用可能。 ただし、予測の信頼区間を解釈する際は注意が必要。

実務での活用ポイント：
  1. 単一の指標だけでなく、複数の指標を総合的に判断
  2. 統計的有意性だけでなく、実務的な意味を考慮
  3. モデルの前提条件を満たしているか確認
  4. 予測の不確実性を認識し、適切な意思決定を行う

⚠️ よくある誤解

  • R²が高い = 良いモデル
    → 過学習の可能性。交差検証で確認必要
  • p値 < 0.05 = 重要な変数
    → 効果の大きさも確認。実務的意味を考慮
  • 相関が高い = 因果関係
    → 相関≠因果。理論的背景の検討が必要
  • 複雑なモデル = 良いモデル
    → シンプルで解釈しやすいモデルが実務では有用

🎯 まとめ：分析の流れ

  1. データの前処理
    • 外れ値の検出と処理
    • 欠損値の確認
  2. 探索的データ分析
    • 基本統計量の確認
    • 相関分析
  3. モデル構築前の確認
    • 多重共線性チェック（VIF）
    • 前提条件の検証
  4. モデル構築と評価
    • 複数手法の比較（線形、正則化、決定木）
    • 交差検証による性能評価
  5. モデル診断
    • 残差分析
    • 影響力のある観測値の確認
  6. 結果の解釈と報告
    • 統計的意味と実務的意味の両面から解釈
    • 限界と改善点の明記
